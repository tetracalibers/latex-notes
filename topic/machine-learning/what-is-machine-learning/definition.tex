\section{人工知能と機械学習}
\marginnote{\refbookMA p4〜6}

\keyword{人工知能}は包括的な用語

\begin{definition}{人工知能}
  コンピュータが決定を下すことができるすべてのタスクを集めたもの
\end{definition}

\keyword{機械学習}は人工知能の一部

\begin{definition}{機械学習}
  コンピュータが「\keyword{データ}に基づいて」決定を下すことができるすべてのタスクを集めたもの
\end{definition}

\keyword{データ}とは、「経験」を表すコンピュータ用語
