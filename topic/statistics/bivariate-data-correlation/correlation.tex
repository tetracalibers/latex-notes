\section{変量の相互関係：相関}
%\marginnote{}

2次元データ$(x,y)$において、2つの変量$x$と$y$の間に相互関係がみられるとき、$x$と$y$の間には\keywordJE{相関}{correlation}関係があるという。

\begin{itemize}
  \item \keyword[Rhodamine]{正の相関}：$x$が大きいほど、$y$も大きくなる傾向がある
  \item \keyword[Cerulean]{負の相関}：$x$が大きいほど、$y$は小さくなる傾向がある
  \item \keyword[Orchid]{無相関}：どちらにも当てはまらない（直線的な関係がない）
\end{itemize}
